To run these executables,

1. Have a working internet connection
2. Install wampserver2.5-Apache-2.4.9-Mysql-5.6.17-php5.5.12-64b.exe   (http://www.filecluster.com/downloads/WampServer.html)
3. Create a new databse with name "oop"
4. Inside the databse, import the "oop.sql" file.
5. Now start the executable "Resource Management Group 18.jar"
